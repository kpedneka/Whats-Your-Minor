# What's Your Minor
---
A web app for you to see which majors and minors you're missing out on.


# Getting started
---
Clone this repository on your local machine
 ``` sh 
 $ cd /path/to/repository
 $ npm install
 $ nodemon
 ```
 App listens on port 3000
 [Go to localhost:3000/](http://localhost:3000/)
